# PS_Sustainability_Software

PS_Sustainability_Software is a frontend-based prototype for managing multi-vendor onboarding with ESG (Environmental, Social, Governance) evaluation.  
It allows vendors to register, complete verification, undergo ESG scoring, and get approved based on risk thresholds.  

The system includes features like multi-vendor management, audit tracking, payment simulation, and ERP-ready Excel export.   

This is a prototype version (no backend), focused on UI/UX and system logic simulation.
